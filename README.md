## Tabs & Accordion

An interactive UI component featuring a tabbed interface with smooth sliding indicator animations and an accordion with expand/collapse functionality. Uses modern glassmorphism styling, dark theme design, and fluid transitions for a polished user experience.

---

## Features

- **Tabbed Navigation** – Switch between Design, Code, and Deploy sections with a smooth sliding indicator
- **Accordion Panel** – Collapsible FAQ-style questions with expand/collapse animations
- **Glassmorphism Design** – Modern dark theme with backdrop blur and subtle borders
- **Smooth Animations** – CSS transitions for fade-in content and rotating icons
- **Responsive Layout** – Adapts to different screen sizes
